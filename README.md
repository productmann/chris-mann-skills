# Chris Mann Skills

Internal Cowork skills for You.com employees.

## Available Skills

### youcom-research
Research any topic using the You.com Research API. Returns deep, multi-step web research with cited sources.

**Installation:**
1. Copy the `skills/youcom-research/` folder to `~/.claude/skills/`
2. Restart Cowork
3. On first use, the skill will walk you through API key setup

**First-time API key setup:**
1. Get your key at [you.com/api](https://you.com/api)
2. Open or create `~/.claude/.env`
3. Add: `YDC_API_KEY=your_key_here`
4. Restart Cowork

## Installing a Skill (Manual)

1. Download or clone this repo
2. Copy the skill folder (e.g. `skills/youcom-research/`) into `~/.claude/skills/`
   - On Mac, open Finder → press **Cmd+Shift+G** → type `~/.claude/skills/`
3. Restart Cowork — the skill is immediately available
